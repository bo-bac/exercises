Your assignment is to create a C# application that protects from potential Hyperlink phishing. The application will track a given source directory on your local Windows-computer.

Each HTML-file that is added to the source directory should be moved to a given target directory with the following rule:

  • HTML files that contain Hyperlinks addresses which do not match the Hyperlink text itself, will be replaced, so that the Hyperlink text will be used as the actual hyperlink address.


For example, try to click on the following link: https://www.paypal.com/il/home. Once done, please schedule a ZOOM meeting to demo your solution.
